## Color Scheme

- **Primary Actions**: Blue (#0088FE)
- **Success/Revenue**: Green (#00C49F)
- **Warning/Low Stock**: Orange (#FF8042)
- **Alerts**: Red for critical items
- **Neutral**: Gray for secondary information

## Interactive Elements

### Buttons
- Date range selectors (7d, 30d, 90d) - Toggle style
- Export buttons - Outline style with download icon
- Tab navigation - Underline active state

### Charts
- **Bar Chart**: Dual-axis (revenue and orders)
  - Hover shows exact values
  - Responsive to screen size
  
- **Pie Chart**: Shows status distribution
  - Click/hover shows percentage
  - Color-coded by status

### Cards
- Rounded corners
- Subtle shadow
- Hover effect for interactive cards
- Badge indicators for important metrics

## Responsive Behavior

### Desktop (>1024px)
- 3-column layout for summary cards
- Side-by-side charts
- Full width tables

### Tablet (768px - 1024px)
- 2-column layout
- Stacked charts
- Scrollable tables

### Mobile (<768px)
- Single column layout
- Simplified charts
- Collapsible sections
- Bottom navigation for tabs

## Export Functionality

When clicking export buttons:
1. Button shows loading state: "Exporting..."
2. API call is made to fetch data
3. Data is converted to Excel format
4. File is automatically downloaded
5. Success toast notification shown
6. File naming: `{type}_export_YYYY-MM-DD.xlsx`

Example filenames:
- `orders_export_2025-10-16.xlsx`
- `customers_export_2025-10-16.xlsx`
- `products_export_2025-10-16.xlsx`

## Loading States

### Initial Load
- Skeleton screens for cards
- Pulsing animation
- Preserves layout structure

### Data Refresh
- Loading spinner overlay
- Disabled interaction during fetch
- Smooth transition to new data

## Error Handling

### API Errors
- Toast notification with error message
- Retry button where applicable
- Graceful degradation (show cached data if available)

### Empty States
- Friendly messages when no data
- Suggestions for next steps
- Appropriate icons

Examples:
- "No sales data available" - when no orders in period
- "All products are well stocked!" - when no low stock items
- "No customers yet" - when no customer records

## Accessibility

- Semantic HTML structure
- ARIA labels for icons
- Keyboard navigation support
- Screen reader friendly
- Sufficient color contrast
- Focus indicators on interactive elements
