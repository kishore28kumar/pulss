# Pulss - Complete User Guide

## Overview

Pulss is a comprehensive white-label e-commerce platform that rivals Amazon and Flipkart, specifically designed for pharmacies, grocery stores, and local businesses. It provides AI-powered search, real-time analytics, and comprehensive business management tools.

## Troubleshooting

### Common Issues
1. **Legal Agreement Not Working**
   - Clear browser cache and cookies
   - Ensure JavaScript is enabled
   - Try in incognito/private mode

2. **Upload Issues**
   - Check file size limits (images: 5MB, CSV: 10MB)
   - Ensure proper file formats (PNG, JPG for images; CSV for data)
   - Verify CSV structure matches template

3. **Payment Issues**
   - Verify UPI ID is correct and active
   - Check bank account details
   - Ensure payment gateway credentials are properly configured

4. **Search Not Working**
   - Check if AI search mappings are configured for business type
   - Verify products have proper keywords and descriptions
   - Ensure search suggestions are active

### Support
- **For Super Admin Issues**: Check platform logs and analytics
- **For Business Issues**: Contact Super Admin support
- **For Customer Issues**: Contact respective business directly via WhatsApp or in-app messaging
